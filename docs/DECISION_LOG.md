# Decision Log

## D-001 — China First
第一阶段只专注中文与中国 Journey，先做深度，不追求国家数量。

## D-002 — No Ads
产品内永不加入第三方广告。

## D-003 — Free Access
所有 Explorer 都可探索所有 Journey；Explorer Pass 仅增加每日探索量。

## D-004 — Chinese First
正文以中文为主，母语、中文解释或双语由 Explorer 自行设置。

## D-005 — Simplified / Traditional
全局提供简体与繁体快捷切换，并记住选择。

## D-006 — Truth Policy
任何内容都不可瞎编；无法确认时明确说明。

## D-007 — Original Art
所有视觉资源必须原创、购买授权或符合可商用开放授权。

## D-008 — Automatic Memory
每次 Journey 自动保存学习记录与感受，不询问是否保存。
