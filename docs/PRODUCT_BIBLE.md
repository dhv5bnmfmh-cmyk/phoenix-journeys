# Phoenix Product Bible v0.1

## Why We Exist

语言不只是考试工具，而是理解一个国家、文化与世界的入口。

Phoenix 希望帮助 Explorer：

- 学会真实、自然的语言
- 对世界产生好奇
- 留下个人学习与人生回忆
- 在学习后愿意继续探索现实世界

## Product Positioning

Phoenix 不是传统课程平台，而是 **Language Journey Platform**。

每段 Journey 固定包含：

1. Story
2. Words
3. Discover
4. Wonder
5. Express
6. Memory
7. World Connection

## Non-negotiable Rules

1. Truth before engagement.
2. Originality before imitation.
3. Explorer before metrics.
4. Simple before complex.
5. Meaningful interaction before passive content.
6. Automatic memory preservation.
7. No advertising.
