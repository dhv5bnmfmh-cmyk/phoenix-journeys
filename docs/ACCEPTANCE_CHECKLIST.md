# Founder Acceptance Checklist

每个版本都由 Chief Explorer 完整体验后验收。

## 30 秒测试

- [ ] 第一次打开就知道产品做什么
- [ ] 首页有目的地文化氛围
- [ ] 开始按钮清楚且有地方特色
- [ ] 页面没有过多选项

## Journey

- [ ] Story 有画面感，不像课本
- [ ] 难度符合 Explorer 当前能力
- [ ] 不认识的字或词可以点击查看
- [ ] 拼音、中文解释和辅助翻译清楚分色
- [ ] Words 有原创图像或明确视觉象征
- [ ] Discover 至少带来一次“原来如此”
- [ ] Explorer 回答后 AI 必须回应
- [ ] AI 建议友善、具体且不制造焦虑
- [ ] Memory 自动保存，不再次询问
- [ ] Passport 地图坐标准确
- [ ] Journey 完成后想继续探索

## 内容与知识产权

- [ ] 事实附有可靠来源
- [ ] 传说和争议内容已标记
- [ ] 没有 AI 编造事实
- [ ] 所有图形、插画、字体与音频均原创或合法授权
